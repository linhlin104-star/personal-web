# Website Thực phẩm chay Khỉ Vàng

## Cài đặt Git

1. Khởi tạo git local:

```
git init
```

2. Thêm remote repository:

```
git remote add origin https://github.com/yourusername/khi-vang.git
```

3. Commit và push code:

```
git add .
git commit -m "Initial commit: Website Thực phẩm chay Khỉ Vàng"
git push -u origin main
```
